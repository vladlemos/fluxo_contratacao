Reunião com StakeHolders - Projeto de desenvolvimento da Esteira de Contratação

Contextualizando...

31/01/2019 - Ocorreu a reunião dos Gestores e equipe CEOPC para escolha do Projeto para inicio no 1º Semestre de 2019

Foram escolhidos: Esteira Contratação (1), Melhoria Siaf(2), Atendimento BNDES (3), Macro SINCE (4).

01/02/2019 ~ 14/02/2019: Foi realizada a primeira SPRINT visando ao entendimento e esboço do processo, para análise de requisitos

22/02/2019 - Fechamento do Escopo do projeto Escolhido.

Equipe de TI: Levantou requisitos de infra-estrutura, bases de dados, linguagens de programação, com apoio GITEC-SP.

É necessário ainda fechar regras de negócio como: mensageria, validação de fases, visão para áreas CAIXA, regras para arquivamento dos documentos, nomenclatura de fases.

25/02/2019 - Levantamento de Requisitos pela equipe de negócios. Prazo 2 Sprints.
Metodologias Aplicadas
  • Kanban: Treinamento de estagiários para atualização semanal do quadro, e nossa atualização online no trello.
  • MVC: separação da regra de negócios das regras de visualização.
  • Time Boxed: Sprints de 14 dias para posicionamento andamento do projeto.
Em sendo aprovado o escopo, como entendemos as Sprints?

A sprint inicial do processo será iniciada pela codificação simultânea de tela de cadastro inicial, layout e interatividade (html/css/js - front-end), regras para banco de dados e classes (codificação php/sql - back-end). Vez que será período de férias de 33% da equipe, com inicio em 25/03.

Propomos refatorar o SIAF nestes 10 dias úteis iniciais.
